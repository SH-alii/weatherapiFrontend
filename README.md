# Weather Frontend

Next.js frontend for the Weather API. Built with React, TypeScript, and Tailwind CSS.

## Installation

1. Install dependencies:

```bash
npm install
```

2. Configure the API endpoint (optional):
   Create a `.env.local` file:

```
NEXT_PUBLIC_API_URL=http://localhost:3000
```

## Running the Frontend

### Development Mode

```bash
npm run dev
```

The frontend will be available at `http://localhost:3000`

### Production Build

```bash
npm run build
npm start
```
